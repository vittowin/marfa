# Марфа и медведь

Готовый статический сайт для GitHub Pages.

## Структура

- `index.html` — главная страница
- `menu.html` — меню и корзина
- `delivery.html` — доставка и самовывоз
- `contacts.html` — контакты
- `css/` — стили
- `js/` — логика меню и корзины
- `images/` — фотографии
- `icons/` — иконки
- `fonts/` — локальные шрифты при необходимости

## Как запустить локально

Откройте `index.html` двойным щелчком.

Для корректной проверки всех страниц удобнее использовать расширение Live Server в Visual Studio Code.

## Как опубликовать на GitHub Pages

1. Создайте публичный репозиторий.
2. Загрузите все файлы и папки из проекта в корень репозитория.
3. Откройте `Settings → Pages`.
4. Выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`.
6. Сохраните настройки.

## Как менять товары

Откройте файл:

`js/menu-data.js`

Каждый товар хранится как отдельный объект:

```js
{
  id: "meat-pie",
  name: "Пирожок с мясом",
  category: "savory",
  categoryLabel: "Сытный",
  description: "Начинка из говядины.",
  price: 58,
  image: "images/products/meat-pie.jpg"
}
```

## Как менять фотографии

Положите фотографию в папку `images/products/` и укажите её путь в `js/menu-data.js`.

Главная фотография:

`images/hero/hero-main.jpg`

## Важно

Ассортимент, цены, режим работы и условия доставки нужно сверить перед публикацией.
